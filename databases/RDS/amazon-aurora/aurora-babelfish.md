# Babelfish for Aurora Postgre SQL

----------

- Allows Aurora PostgreSQL to understand commands targeted for MS SQL Sever (microsoft)
- Therefore microsoft SQL Server based applications can work on Aurora PostgreSQL
- Requires no to little code changes (using the same MS SQL Server client driver)
- The same app can be used after a migration of your database (using AWS SCT (schema conversion tool) and DMS (database migration service))
