### EC2 sizing and config options

1.Operating system: Could be linux, win, or mac os 
2.How much compute power & cores (CPU)
3.How much random access memory (RAM)
4.How much storage space:
  1.network-attached(EBS & EFS)
  2.hardwate(EC2 instance store)
5.Network card: speed of the card, public ip addres
6.Firewall rules: Security group
7.Bootstrap script (configure at first launch): EC2 User  data
