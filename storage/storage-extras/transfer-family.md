# Amazon AWS Transfer Family

- A fully-managed service for file transfers into and out of Amazon S3 or Amazon EFS using the FTP protocol

- Supported Protocols
    * AWS Transfer for FTP (File Transfer Protocol (FTP))
    * AWS Transfer for FTPS (File Transfer Protocol over SSL (FTPS))
    * AWS Transfer for SFTP (Secure File Transfer Protocol (SFTP))

- Managed infrastructure, scalable, reliable, HA (Multi AZ)
- Pay per provisioned endpoint per hour + data transfers in GB
- Store and manage users credential within the service 
- Integrate with existing authentication system (Microsoft Active Directory, LDAP, Okta, Amazon Cognito, Custom)
- Usage: Sharing Files, public datasets, CRM, ERP...
